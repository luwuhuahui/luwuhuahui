# vue-up（Vue3 企业级后台）学习笔记 & 实现记录

## 一、项目简介
- 项目名称：vue-up
- 模板来源：Vue Vben Admin（https://github.com/anncwb/vue-vben-admin）
- 技术栈：
  - Vue 3 + Vite + TypeScript
  - 状态管理（Pinia 或 Vuex，视模板版本而定）
  - UI 组件库（Ant Design Vue 等）
- 主要目标：
  - 熟悉 Vue3 + 组合式 API
  - 学习企业级后台项目的整体架构（布局、路由、权限、状态管理）

## 三、目录结构理解（Vue Vben Admin）

> 建议边看代码边在这里做笔记。

- `src/main.ts`：入口文件，创建应用实例，挂载全局插件
- `src/router/`：路由配置 & 动态路由
- `src/store/` 或 `src/store/modules/`：全局状态管理
- `src/views/`：具体业务页面
- `src/layouts/`：整体布局（顶部导航、侧边菜单、多页签等）
- `src/components/`：通用组件
- `src/api/`：接口定义与请求封装
- `src/hooks/`：组合式 API 封装（自定义 hooks）
- `src/utils/`：工具函数

可以根据自己理解，给每个目录补充：**“实际在项目中看到的例子 + 自己的理解一句话”**。

## 十二、前端交互效果 & UI 设计灵感网站汇总

> 用途：
> - 平时刷这些网站，收藏自己喜欢的页面/交互/组件效果。
> - 在本项目里挑选一个效果，尝试“临摹 + 实现”，然后把过程记录到上面的章节中。

### 1. 原型设计 & 交互流程（类似墨刀）

- **墨刀（MockingBot）**  
  https://modao.cc/  
  用途：低保真/高保真原型设计、页面流程图、简单交互动效。

- **Figma**  
  https://www.figma.com/  
  用途：UI 设计、组件库、原型设计，多人协作；很多开源设计资源可直接拿来参考。

- **Axure RP**  
  https://www.axure.com/  
  用途：更复杂的交互原型与流程设计，在一些传统 B 端项目中仍然常见。

> 建议：
> - 先用墨刀/Figma 画出后台页面的简单原型，再在 Vue 项目中实现对应布局和交互。

### 2. UI 设计灵感 & 效果展示

- **Dribbble**  
  https://dribbble.com/  
  用途：高质量 UI 设计与动效展示，适合找整体风格和组件设计灵感。

- **Behance**  
  https://www.behance.net/  
  用途：完整项目案例展示（Web / App / 品牌），可以学习配色、排版、整体视觉语言。

- **Awwwards**  
  https://www.awwwards.com/  
  用途：创意网站奖项展示，交互和动画很多较炫；适合开眼界、提升交互和动效审美。

- **Lapa Ninja / One Page Love**（可选）  
  搜索关键词：`landing page inspiration`  
  用途：单页网站、Landing Page 的布局与动效灵感。

### 3. 组件与交互效果灵感

- **Ant Design 组件示例**  
  https://ant.design/components/overview-cn/  
  用途：浏览各种业务组件（表格、表单、步骤条、结果页等）的官方交互设计和文案。

- **Element Plus / Element-UI 文档示例**  
  https://element-plus.org/zh-CN/  
  https://element.eleme.cn/#/zh-CN  
  用途：后台常见组件的官方示例，可以直接在项目中按需“搬运 + 优化”。

- **CodePen / CodeSandbox**  
  https://codepen.io/  
  https://codesandbox.io/  
  用途：很多前端开发者分享小组件、小动画、小交互的在线示例，可以 Fork 学习。

### 4. 动效 & 动画灵感

- **LottieFiles**  
  https://lottiefiles.com/  
  用途：丰富的 JSON 矢量动画资源，可在 Web 中无损缩放播放。

- **Animista**  
  https://animista.net/  
  用途：CSS 动画效果生成器，选择效果后直接生成 CSS 代码，可用于按钮、加载动画等。

- **Motion UI / Framer Motion 示例**（偏 React 但可借鉴）  
  关键词：`framer motion examples`  
  用途：学习元素进出场、列表过渡、微交互等动效思路，然后用 Vue 的方式实现。

### 5. 实战型教程与案例站点

- **掘金 / 知乎专栏**  
  https://juejin.cn/  
  用途：搜索关键词如“后台管理系统 实战”、“Vue3 企业后台”、“Vben Admin 实战”等。

- **GitHub Awesome 列表**  
  关键词：`awesome vue`、`awesome frontend`
